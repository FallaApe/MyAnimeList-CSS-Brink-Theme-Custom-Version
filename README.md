## 📥 How to Install

1️⃣ Go to your MAL Anime List  
2️⃣ Click **⚙ List Settings → Theme → CSS**  
3️⃣ Paste the contents of this file:  
👉 `themes/brink-custom-theme.css`  

4️⃣ Save ✅  
5️⃣ Refresh your anime list page

Boom! Your custom theme is applied 🎉
